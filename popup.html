About Hasher

Hasher is an offline developer toolbox. Input is processed locally and is never transmitted, but legacy algorithms are provided only for compatibility and must not be used as security primitives.

  • Hash: MD4, MD5, SHA-1, SHA-2, RIPEMD-160, and Whirlpool
  • HMAC: the same hash families, using UTF-8 input and key bytes
  • CRC: CRC-8, CRC-16, FCS-16, and CRC-32 over UTF-8 bytes
  • Cipher: AES-256-GCM with PBKDF2-HMAC-SHA256 and authenticated output
  • Net: IPv4/subnet calculator, including CIDR /0 through /32
  • Time, numbers, strings, and encoding: strict, lossless developer conversions

Click a Copy button to place a result on the clipboard.

Legacy hashes are powered by hash-wasm 4.12.0. Modern authenticated encryption uses the browser Web Crypto API.
