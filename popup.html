About Hasher

Input is processed locally and is never transmitted. Legacy algorithms remain available only for compatibility and must not be used as security primitives.

Hash & HMACModern and compatibility algorithms
Authenticated cipherAES-256-GCM with PBKDF2
Network toolsIPv4 and CIDR /0 through /32
ConversionsTime, numbers, text, and encoding

Legacy hashes are powered by hash-wasm 4.12.0. Authenticated encryption uses the browser Web Crypto API.
